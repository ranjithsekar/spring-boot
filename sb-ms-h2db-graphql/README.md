##GraphiQL Console
http://localhost:6060/graphiql

##Add Product
http://localhost:6060/graphiql

```json
mutation {
  newProduct(name: "Redmi 8A", category: "Mobile", price: 7000) {
    id
  }
}
```

##Find All Products
http://localhost:6060/graphiql

```json
{
  findAllProducts {
    id name price
  }
}
```

##Find Product By Id
http://localhost:6060/graphiql

```json
{
  findProductById(id: 1) {
    name
  }
}
```
